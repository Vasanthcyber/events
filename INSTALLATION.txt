EVENT MANAGEMENT SYSTEM - QUICK INSTALLATION GUIDE
=================================================

STEP 1: COPY FILES
------------------
Copy the entire 'event_management_system' folder to:
- XAMPP: C:/xampp/htdocs/
- WAMP: C:/wamp/www/
- LAMP: /var/www/html/

STEP 2: CREATE DATABASE
-----------------------
1. Open phpMyAdmin (http://localhost/phpmyadmin)
2. Create new database named: event_management
3. Import the database.sql file from the project folder

STEP 3: CONFIGURE
-----------------
1. Open includes/config.php
2. Update database credentials if needed:
   - DB_HOST: localhost
   - DB_USER: root
   - DB_PASS: (your password)
   - DB_NAME: event_management

STEP 4: SET PERMISSIONS
-----------------------
Make sure assets/images/ folder has write permissions

STEP 5: ACCESS SYSTEM
---------------------
Open browser and go to:
http://localhost/event_management_system/
