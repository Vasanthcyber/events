IMAGES FOLDER
=============

This folder is for uploaded images (hall photos, vendor logos, etc.)

The system will automatically upload images here when you:
- Add a new hall with an image
- Add a vendor with a logo/image

Make sure this folder has write permissions:
- Windows: Right-click → Properties → Security → Edit
- Linux/Mac: chmod 755 assets/images/

Sample images can be added manually for testing.
